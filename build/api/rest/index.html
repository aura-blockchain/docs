REST API

The AURA REST API provides HTTP endpoints for querying blockchain state.

Base URL​

https://testnet-api.aurablockchain.org

Authentication​

No authentication required for read operations.

Endpoints​

Node Info​

GET /cosmos/base/tendermint/v1beta1/node_info

Response:

{
  "default_node_info": {
    "network": "aura-mvp-1",
    "version": "0.38.0"
  },
  "application_version": {
    "name": "aura",
    "version": "1.0.0-mvp"
  }
}

Latest Block​

GET /cosmos/base/tendermint/v1beta1/blocks/latest

Block by Height​

GET /cosmos/base/tendermint/v1beta1/blocks/{height}

Account Balance​

GET /cosmos/bank/v1beta1/balances/{address}

Response:

{
  "balances": [
    {
      "denom": "uaura",
      "amount": "1000000000"
    }
  ]
}

All Balances (Paginated)​

GET /cosmos/bank/v1beta1/balances/{address}?pagination.limit=100

Validators​

GET /cosmos/staking/v1beta1/validators

Query Parameters:

ParameterDescription
statusFilter by status: BOND_STATUS_BONDED, BOND_STATUS_UNBONDED
pagination.limitNumber of results
pagination.offsetOffset for pagination

Single Validator​

GET /cosmos/staking/v1beta1/validators/{validator_addr}

Delegations​

GET /cosmos/staking/v1beta1/delegations/{delegator_addr}

Governance Proposals​

GET /cosmos/gov/v1beta1/proposals

Single Proposal​

GET /cosmos/gov/v1beta1/proposals/{proposal_id}

Transaction by Hash​

GET /cosmos/tx/v1beta1/txs/{hash}

Search Transactions​

GET /cosmos/tx/v1beta1/txs?events={event}

Example:

curl "https://testnet-api.aurablockchain.org/cosmos/tx/v1beta1/txs?events=transfer.recipient='aura1...'"

AURA-Specific Endpoints​

Identity​

GET /aura/identity/v1/did/{did}

Credentials​

GET /aura/vcregistry/v1/credential/{id}

Data Registry​

GET /aura/dataregistry/v1/item/{id}

Broadcasting Transactions​

Simulate​

POST /cosmos/tx/v1beta1/simulate
Content-Type: application/json

{
  "tx_bytes": "base64_encoded_tx"
}

Broadcast​

POST /cosmos/tx/v1beta1/txs
Content-Type: application/json

{
  "tx_bytes": "base64_encoded_tx",
  "mode": "BROADCAST_MODE_SYNC"
}

Modes:

ModeDescription
BROADCAST_MODE_SYNCWait for CheckTx
BROADCAST_MODE_ASYNCReturn immediately
BROADCAST_MODE_BLOCKWait for block inclusion

Error Responses​

{
  "code": 5,
  "message": "not found",
  "details": []
}
CodeDescription
2Unknown error
3Invalid argument
5Not found
11Out of range

Rate Limiting​

  • 100 requests per minute per IP
  • 429 status code when exceeded

SDK Examples​

JavaScript​

const response = await fetch(
  'https://testnet-api.aurablockchain.org/cosmos/bank/v1beta1/balances/aura1...'
);
const data = await response.json();
console.log(data.balances);

Python​

import requests

response = requests.get(
    'https://testnet-api.aurablockchain.org/cosmos/bank/v1beta1/balances/aura1...'
)
print(response.json()['balances'])

curl​

curl -s https://testnet-api.aurablockchain.org/cosmos/bank/v1beta1/balances/aura1... | jq
