# About AURA MVP

The AURA MVP (Minimum Viable Product) is the first public release of the AURA blockchain, focusing on core credential verification functionality.

## Chain Information

| Property | Value |
|----------|-------|
| Chain ID | `aura-mvp-1` |
| Binary | `aurad` |
| Version | `v1.0.0-mvp` |
| Cosmos SDK | v0.50.x |
| CometBFT | v0.38.x |

## What's Included

The MVP includes all essential modules for a functional credential verification network:

- **Core SDK Modules**: auth, bank, staking, slashing, distribution
- **AURA Modules**: identity, vcregistry, dataregistry, compliance, governance, prevalidation
- **Smart Contracts**: CosmWasm (wasm module)

## What's Deferred

The following features are planned for future releases:

- Advanced security modules (security, walletsecurity, validatorsecurity)
- DEX functionality
- Cross-chain bridge
- AI assistant integration
- Advanced monitoring

## MVP Goals

1. **Stability**: Achieve 30+ days of stable testnet operation
2. **Core Functionality**: Prove credential verification use cases
3. **Community**: Enable community participation and feedback
4. **Security**: Complete security audit before mainnet

## Roadmap

- **Q1 2026**: MVP Testnet launch
- **Q2 2026**: Security audit, bug fixes
- **Q3 2026**: Full feature testnet
- **Q4 2026**: Mainnet launch (tentative)
