# AURA Blockchain

Welcome to the official documentation for **AURA Blockchain** - a Cosmos SDK blockchain for credential verification and identity management.

## What is AURA?

AURA provides a decentralized platform for issuing, verifying, and managing credentials. Built on the Cosmos SDK, AURA enables:

- **Credential Issuance**: Organizations can issue verifiable credentials on-chain
- **Credential Verification**: Anyone can verify credentials without contacting the issuer
- **Privacy-Preserving**: Selective disclosure and zero-knowledge proofs
- **Interoperable**: IBC-enabled for cross-chain credential verification
